# Spinner 加载指示器

Spinner 是一个用于表示加载状态的组件，通常用于异步操作过程中提供视觉反馈。

## 引入

```tsx
import { Spinner } from "solarx";
```

## 基础用法

Spinner 组件可以单独使用，也可以嵌入到其他组件中。

```tsx iframe
import { Spinner } from "solarx";

function App() {
  return <Spinner />;
}

export default App;
```

## 不同尺寸

Spinner 组件提供了多种尺寸选项，从超小到超大。

```tsx iframe
import { Spinner } from "solarx";

function App() {
  return (
    <div style={{ display: 'flex', alignItems: 'center', gap: '16px' }}>
      <Spinner size="xs" />
      <Spinner size="sm" />
      <Spinner size="md" />
      <Spinner size="lg" />
      <Spinner size="xl" />
    </div>
  );
}

export default App;
```

## 颜色方案

Spinner 组件提供了多种预设的颜色方案。

```tsx iframe
import { Spinner } from "solarx";

function App() {
  return (
    <div style={{ display: 'flex', alignItems: 'center', gap: '16px' }}>
      <Spinner color="primary" />
      <Spinner color="secondary" />
      <Spinner color="danger" />
      <Spinner color="success" />
    </div>
  );
}

export default App;
```

## 自定义颜色

除了预设的颜色方案，还可以通过 CSS 变量自定义 Spinner 的颜色。

```tsx iframe
import { Spinner } from "solarx";

function App() {
  return (
    <div style={{ display: 'flex', alignItems: 'center', gap: '16px' }}>
      <Spinner style={{ '--spinner-color': 'blue' }} />
      <Spinner style={{ '--spinner-color': 'red' }} />
      <Spinner style={{ '--spinner-color': 'green' }} />
      <Spinner style={{ '--spinner-color': 'purple' }} />
    </div>
  );
}

export default App;
```

## 自定义大小和边框宽度

除了预设的尺寸，还可以通过 CSS 变量自定义 Spinner 的大小和边框宽度。

```tsx iframe
import { Spinner } from "solarx";

function App() {
  return (
    <div style={{ display: 'flex', alignItems: 'center', gap: '16px' }}>
      <Spinner 
        style={{ 
          '--spinner-size': '40px',
          '--spinner-border-width': '4px',
          '--spinner-color': 'teal'
        }} 
      />
      <Spinner 
        style={{ 
          '--spinner-size': '60px',
          '--spinner-border-width': '6px',
          '--spinner-color': 'orange'
        }} 
      />
    </div>
  );
}

export default App;
```

## 与其他组件结合使用

Spinner 组件可以与其他组件结合使用，例如在按钮中显示加载状态。

```tsx iframe
import { Button, Spinner } from "solarx";
import { useState } from "react";

function App() {
  const [isLoading, setIsLoading] = useState(false);
  
  const handleClick = () => {
    setIsLoading(true);
    setTimeout(() => setIsLoading(false), 2000);
  };
  
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '16px' }}>
      <Button isLoading={isLoading} onClick={handleClick}>
        点击加载
      </Button>
      
      <div style={{ display: 'flex', alignItems: 'center', gap: '8px' }}>
        <Spinner size="sm" />
        <span>数据加载中...</span>
      </div>
    </div>
  );
}

export default App;
```

## API

### Spinner Props

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Spinner 的尺寸 |
| `color` | `'primary' \| 'secondary' \| 'danger' \| 'success'` | - | Spinner 的颜色方案 |
| `className` | `string` | - | 自定义类名 |
| `style` | `React.CSSProperties & { '--spinner-color'?: string; '--spinner-size'?: string; '--spinner-border-width'?: string; }` | - | 自定义样式，支持 CSS 变量 |

## 样式定制

Spinner 组件使用 CSS 变量进行样式定制，您可以通过以下 CSS 变量覆盖默认样式：

| CSS 变量 | 描述 | 默认值 |
| --- | --- | --- |
| `--spinner-color` | Spinner 的颜色 | `currentColor` |
| `--spinner-size` | Spinner 的大小 | 根据 size 属性确定 |
| `--spinner-border-width` | Spinner 的边框宽度 | 根据 size 属性确定 |

您可以通过以下方式覆盖这些变量：

1. 在组件上直接设置 style 属性：

```tsx
<Spinner 
  style={{ 
    '--spinner-color': 'blue',
    '--spinner-size': '2rem'
  }}
/>
```

2. 在 CSS 中全局设置：

```css
:root {
  --spinner-color: blue;
  --spinner-size: 2rem;
}
```

3. 针对特定选择器设置：

```css
.custom-spinner {
  --spinner-color: blue;
  --spinner-size: 2rem;
}
```

然后将类名应用到 Spinner 上：

```tsx
<Spinner className="custom-spinner" />
```
