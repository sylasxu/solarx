

# 浏览器支持

Solarx 基于现代化 CSS 框架 Panda，Panda CSS 采用现代 CSS 功能构建，并使用 PostCSS 添加对旧版浏览器的支持。

Solarx 支持主流浏览器的最新、稳定版本，支持以下功能：

- [CSS 变量](https://caniuse.com/css-variables)
- [CSS 层叠层](https://caniuse.com/css-cascade-layers)
- 现代选择器比如：[:where()](https://caniuse.com/mdn-css_selectors_where) 或者 [:is()](https://caniuse.com/css-matches-pseudo)

## 浏览器列表

根据上述标准，支持以下浏览器：
```txt
>= 1%
last 1 major version
not dead
Chrome >= 99
Edge >= 99
Firefox >= 97
iOS >= 15.4
Safari >= 15.4
Android >= 115
Opera >= 73
```

## Polyfills

如果您需要支持旧版浏览器，您可以在 PostCSS 配置中使用以下 polyfill：

- [autoprefixer](https://github.com/postcss/autoprefixer): 使用来自[Can I Use](https://caniuse.com/) 的值将供应商前缀添加到 CSS 规则。
- [postcss-cascade-layers](https://www.npmjs.com/package/@csstools/postcss-cascade-layers): 添加对 CSS 级联层的支持。

以下是在 `postcss.config.js` 中使用这些 polyfill 的文件示例：

```js
module.exports = {
  plugins: [
    '@pandacss/dev/postcss',
    'autoprefixer',
    '@csstools/postcss-cascade-layers'
  ]
}
```
