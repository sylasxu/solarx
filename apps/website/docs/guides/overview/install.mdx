#### 全局安装

开始使用 Solarx 最简单的方法是使用全局安装，这意味着所有组件都从单个包中导入。

按照以下步骤安装所有 Solarx 组件：

<Steps>
      ### 安装软件包
       请在终端中运行以下命令之一：
<Tabs>
  <Tab label="npm">

```md
npm install solarx motion
```

  </Tab>
    <Tab label="yarn">

```md
yarn add solarx motion
```

  </Tab>
  <Tab label="pnpm">

```md
pnpm add solarx motion
```

  </Tab>
</Tabs>
      ### 提升依赖项设置（可选的）
        :::info
注意：此步骤仅适用于使用安装的用户pnpm。如果您使用其他包管理器安装，则可以跳过此步骤。
        :::
