# 介绍

Solarx 是一个 React 的 UI 库，可以帮助你构建美观且易于访问的用户界面，它基于 
Tailwind CSS 和 Motion，并包含一组实用的工具函数。

#### 我为什么要使用 Solarx？

如果您符合以下条件，Solarx 就适合您：

* 厌倦了重复编写数千个类名
* 想要更易读、更易于维护的代码
* 需要用更少的代码更快地设计界面
* 希望跨端拥有一致的设计系统
* 希望使用标准 UI 部件，而无需对每个细节做出设计决策
* 想要黑暗模式和轻松可覆写的主题
* 想要使用基于真实世界 UI 设计原则的设计系统
* 想要同时实现开发速度和定制化

#### 哲学

我个人认为，优秀的设计能够为软件带来巨大的价值。它是我们和互联网陌生用户之间建立信任的主要方法之一。信任对任何业务都至关重要，因为它是用户掏钱成为客户之前首先要评估的因素。

像我们自己会想的那样，用户可能也会问自己的一些问题可能是：

“这家公司靠谱吗？”
“谁在使用他们的软件和服务？”
“我可以把我的个人数据托付给他们吗？”

而糟糕的设计会给团队带来不好的印象，它会给人留下一种懒惰、未完成的、不稳定的印象，

这向用户传递一个信息就是：**团队并不关心用户体验**！

而好的设计说明背后的团队很用心，我们也会本能的认为未来他们应该会有好的发展，会持续迭代，投入心血。

这让我想到“如果他们如此关心这些微小的细节，他们一定非常关心公司的其他事情，包括他们的客户！”。

一个很好的例子就是 https://snappify.com/ 首页，我第一次遇到它是两年前，当时只是简单的美化展示一下代码块，

**我甚至不需要尝试这个产品就可以知道它一定很好**。
