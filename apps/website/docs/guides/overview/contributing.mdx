# 为 Solarx 做贡献

感谢您有兴趣为 Solarx 💖 做出贡献，您太棒了！

## 概述

以下是一些可以帮助改进 Solarx 的方法

- **改进文档**：添加新的演示、修复拼写错误或添加缺失的信息。
- **添加新的演示**：向网站和故事书添加新的组件演示。提交 PR 以apps/compositions/src/examples
- **修复错误**：报告错误、修复错误或添加缺失的功能。
- **贡献代码**：通过打开 Github 讨论来提出新功能，或者找到现有的错误来进行处理。
- **改进代码**：改进代码、修复错误或添加缺失的功能。

:::info

我们欢迎一切贡献，无论大小。

:::
